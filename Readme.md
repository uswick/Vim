My vim settings

Tabs
==========

:tabe == open tab for edit

gt == go to next tab

gT = go to prev tab


Format code
=============

gg=G

Go to file
============

gf = go to file on the cursor


Split File
=============

:sp = horiz

:vsp = vertical

Ctrl + w + up/left/down/right arrow == navigate splits

Ctrl + wT = move split to own tab
